# 最小表示法

对于 $s$ 的两个子串 $A,B$。

设 $k$ 满足：$A[i] = B[j], ... , A[i+k-1] = B[j+k-1]$。

那么如果 $A[i+k] > B[j+k]$ 那么最小表示的首元素一定不在 $A[i]...A[i+k]$ 中。

# 代码
```cpp
// 返回最小表示的首元素的下标
int work(string s) {
  int n = s.size();
  int i, j, k;
  for (i = 0, j = 1, k = 0; i < n && j < n && k < n;) {
    if (s[(i + k) % n] == s[(j + k) % n])
      k++;
    else {
      if (s[(i + k) % n] > s[(j + k) % n])
        i = i + k + 1;
      else
        j = j + k + 1;
      if (i == j) i++;
      k = 0;
    }
  }
  return min(i, j);
}
```
